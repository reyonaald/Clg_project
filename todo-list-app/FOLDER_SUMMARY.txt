═══════════════════════════════════════════════════════════════════════
                    💡 HOW TO USE THE APP
═══════════════════════════════════════════════════════════════════════

📝 ADDING TASKS:
   • Type task in input field
   • Select priority: Low 🟢, Medium 🟡, High 🔴
   • Optionally set due date
   • Click "Add Task" or press Enter

✓ COMPLETING TASKS:
   • Click circular checkbox
   • Enjoy confetti celebration! 🎉
   • Click again to uncomplete

🔍 FILTERING TASKS:
   • Click filter buttons at top
   • Options: All, Active, Done, High, Medium, Low
   • Active filter highlighted in green

🗑️ DELETING TASKS:
   • Click trash icon (🗑️) next to task
   • Task removed immediately

🧹 CLEARING COMPLETED:
   • Button appears when tasks completed
   • Removes all completed tasks at once

                    🌐 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════

GET  /                     → Display all tasks
POST /add                  → Add new task (priority + due date)
POST /delete/<task_id>     → Delete specific task
POST /toggle/<task_id>     → Toggle completion status
POST /clear                → Clear completed tasks
GET  /filter/<type>        → Filter tasks (active/completed/priorities)
POST /edit/<task_id>       → Edit task text
