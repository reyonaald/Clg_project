═══════════════════════════════════════════════════════════════════════
                    💡 HOW TO USE NEW FEATURES
═══════════════════════════════════════════════════════════════════════

1. ADD A TASK WITH PRIORITY:
   • Type your task
   • Select priority from dropdown (Low/Medium/High)
   • Optionally pick a due date
   • Click "Add Task"

2. FILTER YOUR TASKS:
   • Click any filter button at the top
   • See only tasks matching that filter
   • Active filter is highlighted in green

3. COMPLETE A TASK:
   • Click the checkbox
   • Watch the confetti celebration! 🎉
   • Enjoy the glow effect
   • Task moves to completed state

4. CHECK FOR OVERDUE:
   • Look for pulsing red shadows
   • Shaking due dates mean overdue
   • Complete or reschedule them!
